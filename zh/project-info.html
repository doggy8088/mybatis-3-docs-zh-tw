專案資訊 (Project Information)

本文件提供了與本專案相關之資訊的各類文件及連結的概要，所有的內容都是依據本專案之資訊自動產生，透過自動建構工具 Maven 。

概要 (Overview)

文件 (Document) 描述 (Description)
持續整合 (Continuous Integration) 持續整合是以固定頻率為基礎來建構及測試程式碼的過程，此連結列出所有和持續整合過程相關之定義。
依賴 (Dependencies) 此文件列出本專案的所有依賴，並提供每一依賴的相關資訊。
Dependency Information This document describes how to include this project as a dependency using various dependency management tools.
Distribution Management This document provides informations on the distribution management of this project.
問題追蹤 (Issue Tracking) 此為本專案問題管理系統 (Issue Management System) 的連結。可以使用此連結來產生及查詢問題 (issues) (缺陷 (bugs), 功能 (features), 變更需求 (change requests))。
專案授權許可 (Project License) 此為專案授權許可 (Project License) 定義資訊的連結。
郵件列表 (Mailing Lists) 此文件提供本專案郵件列表 (Mailing Lists) 的訂閱 (subscription) 和歸檔 (archive) 的相關資訊。
外掛程式管理 (Plugin Management) 本文件列出定義於 pluginManagement 中的所有外掛程式 (Plugins)。
專案外掛程式 (Project Plugins) 本文件列出本專案所使用之建構外掛程式 (Build Plugins) 及報表外掛程式 (Report Plugins)。
原始碼貯藏庫 (Source Repository) 可藉由 Web 瀏覽器來檢視下列在線原始碼貯藏庫的連結。
專案摘要 (Project Summary) 本文件列出了與本專案相關的其它資訊。
專案團隊 (Project Team) 本文件提供參與本專案之成員相關資訊。這些人員以各種形式的參與對本專案作出貢獻。
